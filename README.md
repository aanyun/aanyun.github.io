- HTML5 Browser

[link](http://50.63.138.117:304/)

## Features

* Sign up
* Log in
* See a list of all their todos
* Create new todos
* Edit a todo
* Mark todos as complete
* Drag to Reorder todos (just client-side)
* Delete a todo (just client-side)
* Filters for Search and showing Incomplete Tasks
